Vietugram сочетает в себе мощность профессиональных мессенджеров с простотой использования.

WebRTC Голосовые чаты
Реальное голосовое общение прямо в браузере. Управление микрофоном, звуком, трансляция экрана с отдельным окном просмотра.
💬
Текстовые чаты
Мгновенный обмен сообщениями, прикрепление изображений, Drag & Drop файлов, редактирование отправленных сообщений.
🔒
Защищённые комнаты
Защита паролем, анонимные комнаты со скрытыми никами, временные комнаты с таймером самоуничтожения.
🤖
Bot API
Подключайте Python-ботов через HTTP API. Каждый бот получает отдельную комнату с особым оформлением.
📌
Закрепление сообщений
Важные сообщения закрепляются вверху чата. Быстрый переход к закреплённому сообщению по клику.

Python Мини-приложения
Встраивайте Python-скрипты прямо в интерфейс. Запускайте во всплывающем окне как приложения в Telegram.

👥 Система ролей

4 уровня доступа

Продуманная иерархия прав с возможностью создавать неограниченное количество кастомных ролей.

👑
Администратор
Полный доступ. Баны, назначения, кастомные сообщения, режим наблюдателя, вход без пароля.
🛡️
Модератор
Удаление комнат и сообщений, кик пользователей, объявления, кастомные роли.
🏆
Ветеран
Автоматически после 1000 сообщений. Зелёный ник и особый статус.
👤
Пользователь
Базовая роль. Создание комнат, чат, изображения, реакции, упоминания.
🏠
Владелец
Создатель комнаты: удаление, закрепление, кик пользователей, защита от спама.
🎨
Кастомная
Создайте свои роли: название, цвет ника, эмодзи-статус, префикс. Назначайте кому угодно.
💎 Почему Vietugram

Лучший выбор
для вашего сообщества

Мы создали Vietugram потому что верим: общение должно быть простым, красивым и мощным одновременно.

⚡

Мгновенная синхронизация

WebSocket соединение через Socket.IO обеспечивает мгновенную доставку сообщений без перезагрузки страницы. История чата сохраняется на сервере.

🎙️

Настоящие голосовые чаты

WebRTC — технология, используемая в Google Meet и Discord. Прямое P2P соединение без промежуточных серверов, минимальная задержка.

🛡️

Надёжная система ролей

Продуманная иерархия: Пользователь → Ветеран → Модератор → Администратор + неограниченное количество кастомных ролей.

🤖

Расширяемость через ботов

Официальный Bot API на Python. Подключайте ботов, автоматизируйте задачи, создавайте интерактивные команды — возможности безграничны.

🎨

Красивый тёмный интерфейс

Современный дизайн с тёмной темой, плавными анимациями и интуитивным UX. Каждая деталь продумана для комфортного использования.

🌐

Простой деплой

Два файла: server.py и index.html. Запустите на своём сервере с белым IP или в Google Colab. Порт 5000, полная инструкция в README.

📱

Работает везде

Открывается в любом современном браузере — на ПК, планшете, смартфоне. Адаптивный дизайн для любого экрана.

🔓

Открытый код

Исходный код доступен на GitHub. Изучайте, модифицируйте, вносите улучшения. Разработчик Vietu открыт к вкладу сообщества.

📋 Как загрузить EXE для скачивания на сервер

1
Положите vietugram_setup.exe в корневую папку проекта рядом с server.py и index.html на своём сервере.
2
Чтобы файл отдавался по кнопке скачивания, добавьте в server.py маршрут:
@app.route('/download') → send_from_directory('.', 'vietugram_setup.exe', as_attachment=True)
3
Либо разместите файл в папке static/ и в index.html укажите ссылку: <a href="/static/vietugram_setup.exe">Скачать</a>
4
Убедитесь что порт 5000 открыт в firewall вашего сервера: sudo ufw allow 5000
5
Запустите сервер: python server.py — сайт доступен по http://ВАШ_IP:5000

# 1. Установка зависимостей pip install flask flask-socketio eventlet # 2. Запуск сервера python server.py # Сайт доступен: # Локально: http://localhost:5000 # По белому IP: http://YOUR_IP:5000 # 3. Открыть порт в firewall (Linux) sudo ufw allow 5000/tcp # 4. Постоянный запуск (фон) nohup python server.py > vietugram.log 2>&1 &
